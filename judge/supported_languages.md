# Ngôn ngữ lập trình được hỗ trợ

Máy chấm FPTOJ hỗ trợ chấm hơn **50 ngôn ngữ lập trình** khác nhau (được cấu hình qua các Executor trong mã nguồn của [FPTOJ Judge Server](https://github.com/FPTOJ-OJ/judge-server)):

Các ngôn ngữ phổ biến gồm có:
- **C/C++**: Hỗ trợ đầy đủ các tiêu chuẩn từ C++03, C++11, C++14, C++17, C++20 (sử dụng trình biên dịch GCC hoặc Clang).
- **Python**: Hỗ trợ Python 3, PyPy 3 (khuyên dùng để cải thiện tốc độ chạy cho Python).
- **Java**: Java 8 và các phiên bản Java mới nhất.
- **Pascal**: Trình biên dịch Free Pascal (FPC) cực kỳ phổ biến trong học đường Việt Nam.
- **Go, Rust, Swift, Kotlin, JavaScript (Node.js)**,... và rất nhiều ngôn ngữ khác.

---

## Mức độ ổn định của các ngôn ngữ

Do đặc thù của thi lập trình thi đấu và học đường, một số ngôn ngữ được sử dụng và kiểm thử nhiều hơn, do đó hoạt động cực kỳ ổn định và tối ưu về mặt giới hạn thời gian (Time Limit) cũng như bộ nhớ (Memory Limit).

Các ngôn ngữ được khuyên dùng hàng đầu cho học sinh gồm:
1. **C++17 / C++20** (Trình biên dịch GCC)
2. **Python 3 / PyPy 3**
3. **Java 8 / Java (latest)**
4. **Pascal (FPC)** (Dành cho các lớp học tin học cơ sở)
